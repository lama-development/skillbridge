# SkillBridge

Nel progetto finale, il codice che si ripete verrà gestito tramite l'uso di **partials** implementati con **Express.js**. Ad esempio, **Navbar** e **Footer** verranno estratti in componenti riutilizzabili (`partials/navbar.ejs` e `partials/footer.ejs`) e inclusi dinamicamente in ogni pagina.

Anche componenti come, ad esempio, le card degli annunci pubblicati saranno generati dinamicamente basandosi sui dati provenienti dal database.

Il progetto finale utilizzerà:
- **Express.js** per le routes e i componenti dinamici
- **JavaScript** per la logica
- **Passport** con **bcrypt** per autenticazione
- **SQLite** come database
- **Bootstrap** come parte grafica con CSS personalizzato

## Istruzioni
Per testare il progetto è necessario semplicemente aprire il file `index.html` situato nella cartella `views` con un qualsiasi browser.
